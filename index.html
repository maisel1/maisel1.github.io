Introduction to the HTML

The HTML (Hyper Text Markup Language) is a computer Markup language used to define the structure of a page.
It is composed by elements, also called "tags" that defines the structure of the page.
To follow an example of element (or tag): <tagname> enter your text here </tagname>.
